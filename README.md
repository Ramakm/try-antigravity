# try-antigravity

## Overview
`try-antigravity` is a demonstration project that showcases advanced agentic coding capabilities. It includes a set of tools and utilities designed to help developers interact with codebases, automate tasks, and generate assets programmatically.

## Features
- Agentic task planning and execution framework
- Integrated tooling for file manipulation, code search, and command execution
- Support for generating images and browser automation
- Structured task boundaries and progress reporting
- Sample workflows and documentation for rapid onboarding

## Getting Started

### Prerequisites
- macOS (or Linux) with a recent version of Node.js and npm
- Python 3.10+ (for auxiliary scripts)
- Git

### Installation
```sh
git clone https://github.com/your-repo/try-antigravity.git
cd try-antigravity
npm install
```

### Running the Agent
```sh
npm run dev
```
This will start the Antigravity agent in interactive mode, ready to accept commands.
